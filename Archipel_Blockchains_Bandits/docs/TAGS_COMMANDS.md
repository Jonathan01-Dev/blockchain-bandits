# Tags Git Requis

Les tags du hackathon doivent pointer sur des commits valides (pas sur un working tree non commit).

## Commandes

```bash
git tag sprint-0
git tag sprint-1
git tag sprint-2
git tag sprint-3
git tag sprint-4
git tag final-submission
```

Push des tags:

```bash
git push origin sprint-0 sprint-1 sprint-2 sprint-3 sprint-4 final-submission
```

Ou tout pousser:

```bash
git push origin --tags
```
